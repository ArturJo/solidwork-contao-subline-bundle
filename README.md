# Contao Subline Bundle

This bundle adds a "Subline" field to the standard headline content element in Contao 5.3+.

## Features

- Adds a `subline` text field to the `tl_content` DCA for headline elements.
- Extends the core Twig template `content_element/headline.html.twig` to display the subline.
- Fully compatible with Contao 5.3 and newer.
